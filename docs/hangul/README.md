## 레슨 유형

### 표준 레슨 흐름
```
Intro (소개) → Sound Explore (소리 탐색) → Sound Match / Quiz MCQ (퀴즈)
→ Syllable Build (조합) → Summary (완료)
```

### 특수 레슨
- **보너스 단어 읽기**: 실제 한국어 단어로 응용 학습
- **미션**: 시간 제한 내 음절 조합 도전
- **Stage Complete**: 스테이지 완료 축하 + 배지

---

## 스텝 타입 11가지

| 스텝 타입 | 설명 | 채점 |
|----------|------|------|
| `intro` | 텍스트+이모지 소개, 개념 설명 | - |
| `soundExplore` | 음절/단어 청취, 입모양 시각화 | - |
| `soundMatch` | 소리 듣고 올바른 문자 선택 | O |
| `quizMcq` | 객관식 퀴즈 (2~4지선다) | O |
| `syllableBuild` | 자음+모음 탭하여 음절 생성 | O |
| `dragDrop` | 드래그 앤 드롭으로 음절 조립 | O |
| `missionIntro` | 미션 설명 (시간, 목표) | - |
| `timedMission` | 시간 제한 음절 조합 게임 | O |
| `missionResults` | 미션 결과 표시 | - |
| `summary` | 레슨 완료, 점수+레몬 보상 | - |
| `stageComplete` | Stage 완료 축하, Master 배지 | - |

점수 = (정답 수 / 전체 문제 수) × 100%
