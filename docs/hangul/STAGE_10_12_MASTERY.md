## Stage 11: 음운 변화 규칙

### 학습 목표
- 5가지 핵심 음운 변화 규칙 이해
- 표기와 실제 발음의 차이 학습
- 규칙 식별 능력 배양

### 레슨 목록 (8개)

---

#### 11-1: 음운 변화 소개

| Step | 타입 | 내용 |
|------|------|------|
| 0 | intro | "한글은 쓰인 대로 읽히지 않는 경우가 많습니다" |
| 1 | soundExplore | 4개 예시 (표기→발음) |
| 2 | summary | 완료 |

**예시 단어**:
| 표기 | 실제 발음 | 적용 규칙 |
|------|----------|----------|
| 학교 | 학꾜 | 경음화 |
| 국물 | 궁물 | 비음화 |
| 좋다 | 조타 | 격음화 |
| 없어 | 업써 | 연음법칙 |

---

#### 11-2: 연음법칙 (Liaison)

**규칙**: 받침 뒤에 모음 → 받침이 다음 음절 초성으로 이동

| Step | 타입 | 내용 |
|------|------|------|
| 0 | intro | 규칙 설명 + 예시 |
| 1 | soundExplore | 5개 단어 |
| 2 | soundMatch | 4문제 |
| 3 | quizMcq | 2문제 |
| 4 | summary | 완료 |

**학습 단어**:
| 표기 | 실제 발음 |
|------|----------|
| 음악 | 으막 |
| 한국어 | 한구거 |
| 없어 | 업써 |
| 읽어 | 일거 |
| 닫아 | 다다 |

**SoundMatch** (4문제): 실제 발음 듣고 선택
| 정답 | 선택지 |
|------|--------|
| 으막 | 으막, 음악, 음앜 |
| 한구거 | 한국어, 한구거, 한쿠거 |
| 업써 | 없어, 업써, 업서 |
| 일거 | 읽어, 일거, 익거 |

**QuizMCQ** (2문제):
| 문제 | 정답 |
|------|------|
| "닫아"의 실제 발음? | 다다 |
| 연음법칙 조건? | 받침 뒤에 모음이 올 때 |

---

#### 11-3: 비음화 (Nasalization)

**규칙**: ㄱ/ㄷ/ㅂ 받침 + ㄴ/ㅁ → ㅇ/ㄴ/ㅁ으로 변화

| 표기 | 실제 발음 | 변화 |
|------|----------|------|
| 국물 | 궁물 | ㄱ→ㅇ |
| 받는 | 반는 | ㄷ→ㄴ |
| 합니다 | 함니다 | ㅂ→ㅁ |
| 작년 | 장년 | ㄱ→ㅇ |
| 십만 | 심만 | ㅂ→ㅁ |

**스텝**: Intro → SoundExplore(5단어) → SoundMatch(4문제) → QuizMCQ(2문제) → Summary

---

#### 11-4: 경음화 (Tensification)

**규칙**: ㄱ/ㄷ/ㅂ 받침 + ㄱ/ㄷ/ㅂ/ㅅ/ㅈ → 된소리(ㄲ/ㄸ/ㅃ/ㅆ/ㅉ)

| 표기 | 실제 발음 | 변화 |
|------|----------|------|
| 학교 | 학꾜 | ㄱ→ㄲ |
| 국밥 | 국빱 | ㅂ→ㅃ |
| 식당 | 식땅 | ㄷ→ㄸ |
| 숙제 | 숙쩨 | ㅈ→ㅉ |
| 꽃집 | 꼳찝 | ㅈ→ㅉ |

**스텝**: Intro → SoundExplore(5단어) → SoundMatch(4문제) → QuizMCQ(2문제) → Summary

---

#### 11-5: 격음화 (Aspiration)

**규칙**: ㅎ + ㄱ/ㄷ/ㅂ/ㅈ → 거센소리(ㅋ/ㅌ/ㅍ/ㅊ)

| 표기 | 실제 발음 | 변화 |
|------|----------|------|
| 좋다 | 조타 | ㅎ+ㄷ→ㅌ |
| 놓고 | 노코 | ㅎ+ㄱ→ㅋ |
| 입학 | 이팍 | ㅂ+ㅎ→ㅍ |
| 축하 | 추카 | ㄱ+ㅎ→ㅋ |
| 급행 | 그팽 | ㅂ+ㅎ→ㅍ |

**스텝**: Intro → SoundExplore(5단어) → SoundMatch(4문제) → QuizMCQ(2문제) → Summary

---

#### 11-6: ㄹ의 비음화와 유음화

**규칙 1 — 유음화**: ㄴ+ㄹ → [ㄹㄹ]
**규칙 2 — ㄹ 비음화**: ㄱ/ㅁ/ㅇ+ㄹ → ㄹ이 ㄴ으로 변함

| 표기 | 실제 발음 | 규칙 |
|------|----------|------|
| 신라 | 실라 | 유음화 |
| 연락 | 열락 | 유음화 |
| 심리 | 심니 | ㄹ 비음화 |
| 능력 | 능녁 | ㄹ 비음화 |
| 편리 | 펼리 | 유음화 |

**스텝**: Intro → SoundExplore(5단어) → SoundMatch(4문제) → Summary

---

## 전체 커리큘럼 완료 후 습득 능력

1. **한글 자모 전체** 인식 및 발음
2. **음절 조합** (초성+중성+종성) 자유 구성
3. **기본/거센소리/된소리** 구분
4. **받침** (단일, 이중, 복합) 읽기
5. **음운 변화 규칙** 5가지 이해
6. **간단한 한국어 문장** 읽기 및 발음
7. **실생활 상황별 표현** (인사, 카페, 길 찾기, 음식 주문)
