# Stage 7~9: 된소리/거센소리 + 받침

> **Stage 7**: 된소리/거센소리 대비 (7레슨) | **Stage 8**: 받침 기초 (10레슨) | **Stage 9**: 받침 확장 (9레슨)

---

## Stage 7: 된소리/거센소리 (5가지 대비 그룹)

### 학습 목표
- 기본/거센소리/된소리 3단계 자음 대비
- 5가지 대비 그룹 청각 식별

### 레슨 목록 (7개)

#### 대비 그룹 레슨 (7-1 ~ 7-5)

| 레슨 | 대비 그룹 | 음절 | 선택지 수 |
|------|----------|------|----------|
| **7-1** | ㄱ / ㅋ / ㄲ | 가, 카, 까 | 3지선다 |
| **7-2** | ㄷ / ㅌ / ㄸ | 다, 타, 따 | 3지선다 |
| **7-3** | ㅂ / ㅍ / ㅃ | 바, 파, 빠 | 3지선다 |
| **7-4** | ㅅ / ㅆ | 사, 싸 | 2지선다 |
| **7-5** | ㅈ / ㅊ / ㅉ | 자, 차, 짜 | 3지선다 |

**표준 스텝 흐름 (4~5단계)**:
```
Intro → SoundExplore(입모양, syllable) → SoundMatch(3문제) → [QuizMCQ] → Summary
```

**7-1만 QuizMCQ 포함** (2문제):
| 문제 | 정답 | 선택지 |
|------|------|--------|
| 거센소리는? | ㅋ | ㄱ, ㅋ, ㄲ |
| 된소리는? | ㄲ | ㄱ, ㅋ, ㄲ |

**SoundMatch 패턴** (각 레슨 3문제):
- 기본 자음 → 3개 선택지 중 정답 선택
- 예: 가 / 카 / 까 중 정답 하나 듣고 선택

---

#### 보너스 + 미션 (7-5W, 7-M)

**7-5W: 단어 읽기**
- **8개 단어**: 까치, 토끼, 뿌리, 따로, 쓰다, 아빠, 오빠, 까마귀
- **스텝**: SoundExplore(word) → SoundMatch(6문제, 3지선다) → Summary

**7-M: 미션**
- 시간: **120초**, 목표: **8개**
- 자음 **9개**: ㄱ, ㅋ, ㄲ, ㄷ, ㅌ, ㄸ, ㅂ, ㅍ, ㅃ
- 모음 4개: ㅏ, ㅓ, ㅗ, ㅜ

## Stage 8: 받침(종성) 기초

### 학습 목표
- **받침 개념** 이해 (초성-중성-종성 구조)
- 기본 받침 7개: ㄴ, ㄹ, ㅁ, ㅇ, ㄱ, ㅂ, ㅅ
- 받침 유무 및 받침 종류 식별

### 레슨 목록 (10개)

#### 8-0: 받침 개념 소개

| Step | 타입 | 내용 |
|------|------|------|
| 0 | intro | 받침이란? (하이라이트 설명) |
| 1 | intro | 🔑 받침의 핵심 개념 |
| 2 | quizMcq | 2문제 (ㄴ/ㄹ 받침 인식) |
| 3 | summary | 완료 |

**Quiz MCQ**:
| 문제 | 정답 | 선택지 |
|------|------|--------|
| 받침 인식 1 | ㄴ | ㄱ, ㅏ, ㄴ |
| 받침 인식 2 | ㄹ | ㅁ, ㅏ, ㄹ |

---

#### 개별 받침 레슨 (8-1 ~ 8-7)

| 레슨 | 받침 | 학습 음절 | 퀴즈 타입 | 비교 패턴 |
|------|------|----------|----------|----------|
| **8-1** | ㄴ | 간, 난, 단 | SoundMatch(2) | 간/가/감, 난/나/날 |
| **8-2** | ㄹ | 말, 갈, 물 | SoundMatch(2) | 말/만/맛, 물/문/무 |
| **8-3** | ㅁ | 감, 밤, 숨 | QuizMCQ(2) | 밤/발/밥, 감/간/갓 |
| **8-4** | ㅇ | 방, 공, 종 | SoundMatch(2) | 방/반/밤, 공/곰/고 |
| **8-5** | ㄱ | 박, 각, 국 | QuizMCQ(2) | 박/밤/발, 국/군/굴 |
| **8-6** | ㅂ | 밥, 집, 숲 | SoundMatch(2) | 밥/밤/반, 집/짐/질 |
| **8-7** | ㅅ | 옷, 맛, 빛 | QuizMCQ(2) | 옷/온/옹, 빛/빈/빔 |

**표준 스텝 (3단계)**:
```
SoundExplore(3음절) → SoundMatch 또는 QuizMCQ(2문제) → Summary
```

**8-4만 Intro 포함** (💡 ㅇ 받침 비음 특수성 설명)

**선택지 설계 원칙**: 받침 없음 vs 해당 받침 vs 다른 받침
- 예: `간`의 선택지 = `가`(받침 없음), `간`(ㄴ 받침), `감`(ㅁ 받침)

**8-M: 미션**
