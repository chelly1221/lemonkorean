# Stage 0: 한글 입문

> **4개 레슨** | 한글의 역사, 블록 조립 원리, 소리 탐색, 청각 퀴즈

---

## 학습 목표
- 한글의 역사와 특징 이해
- 자음+모음 결합 원리 (블록 구조) 습득
- 기초 3자음(ㄱ, ㄴ, ㄷ) + 2모음(ㅏ, ㅗ) 소리 인식
- 음절 청각 식별 능력 배양

---

## 레슨 0-0: 한글 소개

**목표**: 한글의 역사와 특징 소개 (채점 없음)

| Step | 타입 | 내용 |
|------|------|------|
| 0 | intro | 📜 한글의 역사 |
| 1 | intro | 👑 세종대왕의 업적 |
| 2 | intro | 📝 한글 학습의 중요성 |
| 3 | summary | 레슨 완료 |

**특징**: 사용자 입력 없이 텍스트/이모지 읽기만 수행. 각 요소는 순차적 애니메이션(scale, fadeIn, slideY).

---

## 레슨 0-1: 블록 조립 (시각적)

**목표**: 자음+모음 드래그하여 음절 만들기

| Step | 타입 | 내용 | 상호작용 |
|------|------|------|---------|
| 0 | intro | ㄱ + ㅏ = 가 결합 애니메이션 | BlockCombineIntroAnimation |
| 1 | dragDrop | 가 만들기 | ㄱ→자음슬롯, ㅏ→모음슬롯 |
| 2 | dragDrop | 나 만들기 (힌트: ㄴ + ㅏ = 나) | 드래그 + 힌트 표시 |
| 3 | dragDrop | 다 만들기 (힌트: ㄷ + ㅏ = 다) | 드래그 + 힌트 표시 |
| 4 | summary | 채점 결과 (3문제) | 레몬 보상 |

### 드래그 드롭 상세
- **문자 풀**: 자음 [ㄱ, ㄴ, ㄷ], 모음 [ㅏ, ㅗ]
- **드롭 영역**: 자음(파랑) + 모음(빨강) 2개 슬롯
- **정답**: 가벼운 진동 + TTS 재생 + 음절 확대 애니메이션 → 1.2초 후 자동 진행
- **오답**: 중진동 + 빨간 테두리 0.4초 → 리셋

---

## 레슨 0-2: 소리 탐색 (청각)

**목표**: 자음, 모음, 음절의 소리 직접 듣기 (채점 없음)

| Step | 타입 | 내용 | 대상 |
|------|------|------|------|
| 0 | intro | 👂 소개 | - |
| 1 | soundExplore | 자음 탐색 | ㄱ→가, ㄴ→나, ㄷ→다 |
| 2 | soundExplore | 모음 탐색 | ㅏ, ㅗ |
| 3 | soundExplore | 음절 탐색 | 가, 나, 다 |
| 4 | summary | 완료 | - |

### Sound Explore 상세
- 중앙 120×120 원형 버튼 탭 → TTS 재생
- 다중 문자: 모든 문자 듣기 완료해야 "다음" 버튼 활성화
- 미청취 문자: 빨간 배지 표시
- 자음/모음: 입모양 애니메이션(MouthAnimationWidget) 표시

---

## 레슨 0-3: 듣고 선택하기 (청각 퀴즈)

**목표**: 음성을 듣고 올바른 음절 선택

| Step | 타입 | 내용 | 문제 수 |
|------|------|------|---------|
| 0 | intro | 🎧 소개 | - |
| 1 | soundExplore | 5개 음절 청취 | 가, 나, 다, 고, 노 |
| 2 | soundMatch | 1차 퀴즈 | 4문제 |
| 3 | soundMatch | 2차 퀴즈 | 4문제 |
| 4 | summary | 채점 결과 (총 8문제) | 레몬 보상 |
| 5 | stageComplete | 🎊 Stage 0 완료! | Stage Master 배지 |

### Sound Match 퀴즈 상세

**1차 (Step 2):**
| 정답 | 선택지 |
|------|--------|
| 가 | 가, 나 |
| 다 | 나, 다 |
| 고 | 고, 노 |
| 노 | 가, 노 |

**2차 (Step 3):**
| 정답 | 선택지 |
|------|--------|
| 가 | 가, 고 |
| 고 | 가, 고 |
| 나 | 나, 노 |
| 노 | 나, 노 |

### Sound Match 상호작용
- 스피커 아이콘(100×100, 파랑) 탭 → 정답 TTS 자동 재생
- 2개 선택지 2×1 그리드(100×100)
- 정답: 녹색 배경 + 가벼운 진동
- 오답: 빨강 배경 + 중간 진동 + 정답 녹색 표시
