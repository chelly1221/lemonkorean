# 확장 모음 스테이지 추가 + 커리큘럼 번호 조정

## 변경 배경

기존 한글 커리큘럼(Stage 0~8)에서 21개 한글 모음 중 **15개가 누락**되어 있었음:
- Stage 1이 기본 모음 6개(ㅏ,ㅓ,ㅗ,ㅜ,ㅡ,ㅣ)만 다루고 이후 스테이지에서 나머지 모음을 소개하지 않음

빠진 모음:
- **Y-모음**: ㅑ, ㅕ, ㅛ, ㅠ
- **ㅐ/ㅔ 계열**: ㅐ, ㅔ, ㅒ, ㅖ
- **복합모음**: ㅘ, ㅙ, ㅚ, ㅝ, ㅞ, ㅟ, ㅢ

## 새 커리큘럼 구조 (11 스테이지)

| # | 제목 | 변경 |
|---|------|------|
| 0 | 한글 구조 이해 | 그대로 |
| 1 | 기본 모음 | 그대로 |
| **2** | **Y-모음 (ㅑ,ㅕ,ㅛ,ㅠ)** | **신규 (7 레슨)** |
| **3** | **ㅐ/ㅔ 모음 (ㅐ,ㅔ,ㅒ,ㅖ)** | **신규 (7 레슨)** |
| 4 | 기본 자음 | 기존 2→4 |
| 5 | 본격 조합 훈련 + 복합모음 | 기존 3→5, **복합모음 3레슨 추가 (6→9)** |
| 6 | 된소리/거센소리 | 기존 4→6 |
| 7 | 받침 1차 | 기존 5→7 |
| 8 | 받침 확장 | 기존 6→8 |
| 9 | 복합 받침 | 기존 7→9 |
| 10 | 단어 읽기 | 기존 8→10 |

## 교육학적 근거

- **Y-모음(Stage 2)**: 기본 모음 직후가 표준 순서 — 획 하나 추가 패턴으로 자연스러운 확장
- **ㅐ/ㅔ(Stage 3)**: 자음 전에 모든 단모음 완료 필요, ㅐ/ㅔ 구분은 집중 훈련 필요
- **복합모음**: 독립 스테이지 대신 조합 훈련(Stage 5)에 통합 — 자음과 함께 맥락에서 학습

## 영향 범위

- Stage 0 인터랙티브 레슨: 영향 없음 (lessonId `0-*` 불변)
- Stages 1~10: 아직 구현 전 (계획 UI만 존재), 진도 데이터 충돌 없음
- `hangul_provider.dart`: Stage 0만 구현되어 변경 불필요
