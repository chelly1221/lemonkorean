# 음성 대화방 (Voice Rooms) 종합 프로덕트 디자인 리뷰

## 현재 구현 상태 요약

### 구축 완료된 기능
- **방 목록**: 활성 방 리스트, Pull-to-refresh, 참여자 수 표시
- **방 생성**: 제목, 주제(선택), 언어 레벨(all/beginner/intermediate/advanced), 최대 스피커 수(2-4)
- **역할 시스템**: Creator, Speaker, Listener 3단계 구조
- **스테이지**: Flame 2D 게임엔진 기반 픽셀 캐릭터, 장비 렌더링, 걷기 이동, 방향 전환
- **채팅**: 임시(ephemeral) 텍스트 메시지, 방 종료 시 삭제
- **리액션**: 8종 이모지 + 5종 캐릭터 제스처(wave, bow, dance, jump, clap)
- **스테이지 관리**: 손들기 → 호스트 승인 → 승격, 강등, 추방, 초대
- **오디오**: LiveKit WebRTC (에코 캔슬, 노이즈 서프레션, AGC, 하이패스 필터)
- **연결 복원**: 지수 백오프 기반 자동 재연결 (최대 3회)
- **네트워크 최적화**: 위치 업데이트 10Hz 스로틀링, 최소 이동량 필터링

### 기술 아키텍처
- Frontend: Flutter (Provider 상태관리) + Flame 게임엔진
- Backend: Node.js Express + PostgreSQL + LiveKit + Socket.IO
- 실시간 통신: Socket.IO (채팅, 위치, 리액션, 제스처), LiveKit (오디오)

## 2. 방 유형 분류 (Room Type Taxonomy)

현재 단일 유형의 방만 지원하고 있어, 학습 목적별 특화된 방 유형 체계가 필요하다.

### 2.1 자유 대화 (Free Conversation)

**설명**: 현재 기본 유형과 유사하지만, 대화 주제 카드와 아이스브레이커를 추가하여 대화가 끊기지 않도록 지원한다.

**사용자 스토리**: "한국어 학습자로서, 부담 없이 자유롭게 한국어로 대화하고 싶다. 대화가 끊기면 주제 카드가 새 화제를 제안해주어 어색한 침묵 없이 계속 연습할 수 있다."

**UI/UX 변경점**:
- 스테이지 상단에 현재 대화 주제 카드 표시 (호스트가 넘김)
- "다음 주제" 버튼 (호스트/스피커 모두 사용 가능)
- 주제 카테고리: 일상, 여행, 음식, 취미, 시사 등
- 남은 대화 시간 옵셔널 타이머

**복잡도**: M | **임팩트**: High | **우선순위**: P0

### 2.2 발음 연습 (Pronunciation Practice)

**설명**: 호스트가 목표 발음을 제시하면, 참여자가 차례대로 발음하고 피드백을 받는 구조. 한글 커리큘럼의 스테이지별 음소와 연동된다.

**사용자 스토리**: "초급 학습자로서, ㄱ/ㅋ/ㄲ 같은 한국어 고유 음소의 발음을 원어민에게 직접 교정받고 싶다. 내 차례가 오면 마이크가 자동으로 켜지고, 피드백을 받은 후 다음 사람으로 넘어간다."

**UI/UX 변경점**:
- "차례 대기열" UI: 발언 순서 표시 + 자동 마이크 제어
- 목표 발음 카드 (한글 글자 + 로마자 표기 + 음성 예시)
- 호스트의 "잘했어요/다시 해봐요" 퀵 피드백 버튼
- 스테이지에 현재 발음 중인 글자를 크게 표시
- 한글 커리큘럼 Stage 0-11 연동: 학습 중인 음소 자동 제안

**복잡도**: L | **임팩트**: Critical | **우선순위**: P0

### 2.3 역할극 (Role-play Scenarios)

**설명**: 미리 정의된 시나리오(카페 주문, 길 묻기, 면접 등)에서 역할을 배정받고 대화하는 방. 존댓말/반말 전환 연습에 특히 유용하다.

**사용자 스토리**: "중급 학습자로서, 실제 상황에서 사용하는 한국어를 역할극으로 연습하고 싶다. '편의점에서 물건 사기' 시나리오를 선택하면, 나는 손님 역할, 상대방은 점원 역할을 맡아 자연스러운 대화를 연습할 수 있다."

**UI/UX 변경점**:
- 시나리오 선택 화면 (카테고리: 일상, 비즈니스, 여행, 학교 등)
- 역할 배정 UI: 각 참여자에게 역할 카드 표시
- 시나리오 스크립트 힌트 (접었다 펼 수 있는 가이드)
- 스테이지 배경이 시나리오에 맞게 변경 (카페, 지하철, 사무실 등)
- 핵심 표현 목록 사이드 패널

**복잡도**: XL | **임팩트**: High | **우선순위**: P1

### 2.4 Q&A 세션 (Q&A Sessions)

**설명**: 한국어/한국 문화에 대한 질문을 올리고, 원어민이나 고급 학습자가 답변하는 구조화된 Q&A 방.

**사용자 스토리**: "한국어 학습자로서, '이/가와 은/는의 차이가 뭐예요?' 같은 문법 질문을 실시간으로 물어보고 음성으로 설명을 듣고 싶다. 채팅으로 질문을 올리면 호스트가 순서대로 답변해준다."

**UI/UX 변경점**:
- 질문 대기열 UI (채팅에서 "질문" 태그 추가 가능)
- 호스트가 질문을 선택하면 스테이지에 핀 표시
- 현재 다루고 있는 질문이 스테이지 상단에 크게 표시
- 질문자를 임시 스피커로 자동 승격하는 옵션
- 유용한 답변에 대한 "도움이 됐어요" 리액션

**복잡도**: M | **임팩트**: Medium | **우선순위**: P2

### 2.5 듣기 이해 (Listening Comprehension)

**설명**: 호스트가 한국어 텍스트를 읽어주고, 청취자가 이해도 퀴즈에 답하는 교육적 방. 뉴스 기사, 동화, 대화문 등을 활용한다.

**사용자 스토리**: "중급 학습자로서, 원어민의 자연스러운 한국어를 듣고 이해도를 확인하고 싶다. 호스트가 뉴스 기사를 읽어주면, 이후 나오는 이해도 퀴즈에 채팅으로 답하며 실력을 확인할 수 있다."

**UI/UX 변경점**:
- 읽기 자료 표시 패널 (호스트만 볼 수 있는 텍스트)
- 퀴즈 팝업 (호스트가 트리거, 참여자가 선택지 탭)
- 정답률 실시간 표시
- 속도 조절 요청 버튼 ("천천히 해주세요" 리액션)
- 자막 on/off 토글 (학습 레벨별)

**복잡도**: L | **임팩트**: High | **우선순위**: P1

### 2.6 토론/디베이트 (Debate/Discussion)

**설명**: 주제에 대해 찬반 또는 다양한 의견을 나누는 구조화된 토론 방. 고급 학습자 대상.

**사용자 스토리**: "고급 학습자로서, 한국어로 논리적인 토론을 연습하고 싶다. '한국에서 가장 살기 좋은 도시는?' 같은 주제로 각자 의견을 말하고, 타이머가 발언 시간을 공평하게 관리해준다."

**UI/UX 변경점**:
- 발언 타이머 (개인별 남은 시간 표시)
- 찬/반 팀 분리 표시 (스테이지 좌/우 배치)
- 투표 기능 (토론 후 청중 투표)
- 핵심 논점 메모 패널
- 스피커 순서 관리 (라운드 로빈 or 자유)

**복잡도**: L | **임팩트**: Medium | **우선순위**: P2

## 3. 게이미피케이션 연동 (Gamification Integration)

현재 앱은 레몬 보상, 픽셀 캐릭터, 장비, 펫 시스템 등 풍부한 게이미피케이션을 보유하고 있으나, 음성 대화방과의 연동이 전혀 없다.

### 3.1 XP/레몬 보상 시스템

**기능 설명**: 음성 대화방 참여 시간과 활동 기반으로 레몬(앱 내 화폐) 보상을 지급한다. 스피커로 활발히 참여할수록 더 많은 보상을 받는다.

**보상 구조 제안**:
| 활동 | 레몬 보상 |
|------|----------|
| 방 참여 (5분 이상) | 1 레몬 |
| 스피커로 10분 대화 | 3 레몬 |
| 방 호스트 (30분 이상) | 5 레몬 |
| 발음 연습 방 완료 | 2 레몬 |
| 역할극 완주 | 4 레몬 |
| 첫 음성 대화방 참여 (일일) | 2 레몬 보너스 |

**사용자 스토리**: "학습자로서, 음성 대화방에 참여하면 레몬을 받고 싶다. 이 레몬으로 캐릭터 장비를 구매할 수 있어서, 대화 연습을 더 자주 하게 된다."

**복잡도**: M | **임팩트**: Critical | **우선순위**: P0

### 3.2 음성 대화방 연속 참여 스트릭

**기능 설명**: 매일 음성 대화방에 참여하면 연속 참여 일수가 증가하고, 마일스톤 달성 시 특별 보상을 제공한다.

**마일스톤 제안**:
- 3일 연속: "첫 발걸음" 뱃지 + 5 레몬
- 7일 연속: "대화 습관" 뱃지 + 15 레몬
- 30일 연속: "한국어 토커" 뱃지 + 50 레몬 + 한정 장비
- 100일 연속: "레전드 스피커" 뱃지 + 200 레몬 + 레어 장비

**사용자 스토리**: "학습자로서, 매일 음성 대화방에 참여하면 연속 참여 기록이 유지되고 특별 보상을 받고 싶다. 이것이 매일 한국어를 말하는 습관을 만드는 동기부여가 된다."

**복잡도**: S | **임팩트**: High | **우선순위**: P0

### 3.3 음성 대화방 업적/뱃지

**기능 설명**: 음성 대화방에서의 다양한 활동에 대한 업적 시스템.

**업적 제안**:
| 업적 이름 | 조건 | 보상 |
|-----------|------|------|
| 첫 마이크 | 첫 스피커 참여 | "마이크" 뱃지 |
| 호스트 데뷔 | 첫 방 생성 | "호스트" 뱃지 |
| 인기 호스트 | 10명 이상 참여 방 호스트 | 레어 모자 장비 |
| 다국적 대화 | 3개 이상 언어 사용자와 대화 | "글로벌" 뱃지 |
| 발음 마스터 | 발음 연습 방 50회 참여 | 특수 이펙트 |
| 끝까지 | 1시간 이상 대화 | "끈기" 뱃지 |
| 도우미 | Q&A에서 10회 답변 | "선생님" 뱃지 |

**사용자 스토리**: "학습자로서, 음성 대화방에서 다양한 활동을 할 때마다 업적을 달성하고 싶다. 프로필에 뱃지를 표시해서 다른 학습자에게 나의 활동 수준을 보여줄 수 있다."

**복잡도**: M | **임팩트**: High | **우선순위**: P1

### 3.4 캐릭터 표현과 음성 활동 연동

**기능 설명**: 음성 대화방에서의 활동이 스테이지 위 픽셀 캐릭터에 실시간으로 반영된다.

**구체적 연동 제안**:
- **말하기 시각화**: 마이크 입력 볼륨에 따라 캐릭터 위 오라(aura) 크기/색상 변화 (현재 기본 구현됨, 확장 필요)
- **활발한 대화 이펙트**: 일정 시간 이상 연속 발언 시 캐릭터 주변에 별/불꽃 파티클
- **음성 대화방 전용 장비**: 마이크, 헤드셋, DJ 모자 등 음성 활동 관련 장비
- **커넥션 품질 시각화**: 현재 구현된 ConnectionQualityBadge를 더 직관적으로 개선
- **레벨 표시**: 캐릭터 위에 한국어 레벨(초급/중급/고급) 뱃지

**사용자 스토리**: "학습자로서, 음성 대화방에서 활발하게 말할수록 내 캐릭터가 더 화려하게 보이길 원한다. 다른 참여자가 내 캐릭터를 보고 '이 사람은 활발한 참여자구나'라고 알 수 있다."

**복잡도**: M | **임팩트**: Medium | **우선순위**: P1

### 3.5 펫 시스템 연동

**기능 설명**: 현재 앱의 펫 시스템을 음성 대화방 스테이지에도 통합하여, 스피커의 펫이 캐릭터 옆에 함께 표시된다.

**사용자 스토리**: "학습자로서, 열심히 키운 내 펫이 음성 대화방 스테이지에서도 내 캐릭터 옆에 보이길 원한다. 다른 참여자들과 펫에 대해 이야기하는 것도 좋은 대화 주제가 된다."

**복잡도**: M | **임팩트**: Low | **우선순위**: P3

## 4. 호스트/크리에이터 도구 강화 (Creator Tools Enhancement)

### 4.1 대화 주제 카드/프롬프트 시스템

**기능 설명**: 호스트가 미리 준비된 대화 주제 카드를 넘기며 대화를 이끌 수 있는 도구. 레벨별로 분류된 주제 카드 라이브러리를 앱 내장으로 제공한다.

**주제 카드 카테고리**:
- 아이스브레이커: "이름이 뭐예요?", "어디에서 왔어요?"
- 일상: "오늘 뭐 했어요?", "주말에 뭐 할 거예요?"
- 문화: "좋아하는 한국 음식이 뭐예요?", "한국 드라마 추천해 주세요"
- 상황극: "카페에서 주문하기", "택시 타기"
- 문법 포인트: "~(으)ㄹ 수 있다 사용해서 문장 만들기"

**사용자 스토리**: "호스트로서, 대화가 끊기지 않도록 미리 준비된 주제 카드를 하나씩 꺼내서 보여주고 싶다. 참여자의 레벨에 맞는 주제가 자동으로 제안되면 더 좋겠다."

**복잡도**: M | **임팩트**: Critical | **우선순위**: P0

### 4.2 발언 시간 타이머

**기능 설명**: 각 스피커의 발언 시간을 관리하는 타이머. 공평한 참여를 보장하고, 특히 토론/발음 연습에서 유용하다.

**모드**:
- **자유 모드**: 타이머 없음 (현재 기본)
- **순서 모드**: 정해진 순서대로 각자 N초/분씩 발언
- **공평 모드**: 총 발언 시간이 가장 적은 사람에게 우선권

**사용자 스토리**: "호스트로서, 발음 연습 방에서 각 참여자에게 공평하게 30초씩 발언 기회를 주고 싶다. 타이머가 자동으로 다음 사람에게 넘겨서 모든 참여자가 연습할 수 있다."

**복잡도**: M | **임팩트**: High | **우선순위**: P1

### 4.3 어휘 공유 패널

**기능 설명**: 대화 중 등장하는 새 어휘나 표현을 호스트가 실시간으로 공유 패널에 추가할 수 있는 기능. 참여자는 이를 자신의 단어장에 저장할 수 있다.

**사용자 스토리**: "호스트로서, 대화 중 새로운 단어나 표현이 나오면 즉시 공유 패널에 추가하고 싶다. 참여자들이 방을 나갈 때 배운 어휘 목록을 가져갈 수 있어서 학습 효과가 높아진다."

**UI 구성**:
- 채팅 영역 옆 슬라이드 패널
- 단어 카드: 한국어 + 발음 + 의미 (참여자 모국어)
- "내 단어장에 저장" 버튼
- 방 종료 시 "오늘 배운 단어" 요약 화면

**복잡도**: L | **임팩트**: Critical | **우선순위**: P0

### 4.4 녹음/다시듣기

**기능 설명**: 방의 오디오를 녹음하여 나중에 다시 들을 수 있는 기능. 자신의 발음을 복습하거나 놓친 내용을 확인할 수 있다.

**사용자 스토리**: "학습자로서, 음성 대화방에서 나눈 대화를 나중에 다시 듣고 싶다. 특히 내 발음을 객관적으로 들어보며 개선점을 찾고 싶다."

**구현 고려사항**:
- LiveKit 서버 사이드 녹음 기능 활용
- 개인정보 보호: 녹음 시작 시 모든 참여자에게 알림 + 동의
- 저장 공간: MinIO에 최대 7일 보관 후 자동 삭제
- 접근 제한: 참여했던 사용자만 녹음 접근 가능

**복잡도**: XL | **임팩트**: High | **우선순위**: P2

### 4.5 공동 호스트 (Co-Host)

**기능 설명**: 호스트가 다른 스피커를 공동 호스트로 지정하여, 스테이지 관리 권한을 공유할 수 있는 기능.

**사용자 스토리**: "호스트로서, 원어민 친구를 공동 호스트로 지정해서 함께 방을 운영하고 싶다. 내가 잠시 자리를 비워도 공동 호스트가 스테이지를 관리할 수 있다."

**공동 호스트 권한**:
- 손들기 요청 승인/거절
- 스피커 강등 (원래 호스트 제외)
- 주제 카드 넘기기
- 참여자 뮤트

**복잡도**: M | **임팩트**: Medium | **우선순위**: P2

### 4.6 자동 중재 (Auto-Moderation)

**기능 설명**: 부적절한 언어 사용 감지, 장시간 뮤트 스피커 자동 알림, 비활성 참여자 관리 등.

**사용자 스토리**: "호스트로서, 방에서 부적절한 발언이 있을 때 자동으로 경고가 나가고, 장시간 뮤트 상태인 스피커에게 발언을 유도하는 알림이 가면 좋겠다."

**복잡도**: L | **임팩트**: Medium | **우선순위**: P3

## 5. 디스커버리 & 매칭 (Discovery & Matching)

### 5.1 방 예약/스케줄링

**기능 설명**: 미래 시간에 열릴 방을 미리 예약하고, 관심 있는 사용자가 알림 등록할 수 있는 기능.

**사용자 스토리**: "호스트로서, '매주 수요일 오후 8시 한국어 자유 대화' 방을 예약하고 싶다. 관심 있는 학습자들이 미리 알림을 등록해서, 시간이 되면 자동으로 참여 알림을 받는다."

**UI 구성**:
- 방 목록에 "예정됨" 섹션 추가
- 달력 뷰 (주간/월간)
- 알림 등록 버튼 ("알림 받기")
- 반복 스케줄 설정 (매일/매주/맞춤)
- 시간대 자동 변환 (글로벌 사용자 지원)

**복잡도**: L | **임팩트**: Critical | **우선순위**: P0

### 5.2 원어민 표시 (Native Speaker Indicator)

**기능 설명**: 방 목록과 참여자 프로필에 원어민 여부를 명확히 표시하여, 학습자가 원어민이 있는 방을 쉽게 찾을 수 있도록 한다.

**사용자 스토리**: "학습자로서, 원어민이 호스트인 방을 우선적으로 찾고 싶다. 방 카드에 '원어민 호스트' 뱃지가 있으면 바로 알 수 있다."

**구현 방법**:
- 프로필 설정에서 모국어 등록
- 방 카드에 원어민 뱃지 (한국 국기 아이콘)
- 필터: "원어민 호스트 방만 보기"
- 스테이지에서 원어민 참여자 특별 표시

**복잡도**: S | **임팩트**: Critical | **우선순위**: P0

### 5.3 태그/카테고리 필터

**기능 설명**: 방에 태그를 추가하고, 사용자가 관심 태그로 필터링할 수 있는 기능.

**태그 예시**:
- 레벨: #초급 #중급 #고급
- 주제: #일상대화 #비즈니스한국어 #TOPIK #K-드라마
- 유형: #발음연습 #역할극 #자유대화 #Q&A
- 언어: #영어OK #일본어OK #중국어OK

**사용자 스토리**: "학습자로서, 내 관심 주제와 레벨에 맞는 방을 빠르게 찾고 싶다. '#중급 #일상대화 #영어OK' 태그로 필터링하면 딱 맞는 방을 찾을 수 있다."

**복잡도**: S | **임팩트**: High | **우선순위**: P0

### 5.4 알림 설정

**기능 설명**: 팔로우한 호스트가 방을 열 때, 관심 태그의 새 방이 생길 때, 예약한 방 시작 전 알림을 받는 기능.

**사용자 스토리**: "학습자로서, 좋아하는 호스트가 새 방을 열면 바로 알림을 받고 싶다. 또한 '#발음연습 #초급' 방이 새로 열리면 자동 알림을 받아서 놓치지 않겠다."

**복잡도**: M | **임팩트**: High | **우선순위**: P1

### 5.5 스마트 매칭

**기능 설명**: 사용자의 학습 레벨, 관심 주제, 활동 시간대, 모국어 등을 기반으로 적합한 방을 추천하는 알고리즘.

**사용자 스토리**: "초급 학습자로서, 앱이 내 레벨과 관심사에 맞는 방을 자동으로 추천해주면 좋겠다. '당신에게 맞는 방' 섹션에서 바로 참여할 수 있다."

**복잡도**: L | **임팩트**: Medium | **우선순위**: P2

## 6. 리텐션 메커니즘 (Retention Mechanics)

### 6.1 정기 방/스케줄

**기능 설명**: 반복적으로 열리는 정기 방을 구독하면, 매번 알림을 받고 자동으로 참여 목록에 추가된다.

**사용자 스토리**: "학습자로서, '매주 화/목 저녁 한국어 일상 대화' 정기 방을 구독하고 싶다. 구독하면 매번 시작 10분 전에 알림을 받아서 잊지 않고 참여할 수 있다."

**복잡도**: M | **임팩트**: Critical | **우선순위**: P1

### 6.2 호스트 팔로우 & 프로필

**기능 설명**: 마음에 드는 호스트를 팔로우하고, 호스트의 프로필에서 향후 방 일정, 과거 운영 기록, 평점 등을 확인할 수 있다.

**호스트 프로필 표시 정보**:
- 총 호스팅 횟수 / 총 참여자 수
- 평균 방 지속 시간
- 주로 다루는 주제/레벨
- 참여자 평점 (별점)
- 다음 예정 방 일정
- 원어민 여부

**사용자 스토리**: "학습자로서, 좋았던 호스트를 팔로우하고 그 호스트의 다음 방 일정을 확인하고 싶다. 호스트 프로필에서 과거 기록과 평점을 보고 참여 여부를 결정할 수 있다."

**복잡도**: M | **임팩트**: High | **우선순위**: P1

### 6.3 방 히스토리 & 통계

**기능 설명**: 사용자의 음성 대화방 활동 통계를 시각화하여 보여주는 기능.

**통계 항목**:
- 이번 주/이번 달 총 대화 시간
- 스피커로 발언한 시간 비율
- 참여한 방 유형 분포
- 대화 레벨 추이 (시간에 따른 참여 방 레벨 변화)
- 가장 활발했던 시간대
- 함께 대화한 사람 수

**사용자 스토리**: "학습자로서, 이번 달 음성 대화방에서 총 몇 시간 한국어를 말했는지 확인하고 싶다. '이번 주 5시간 대화, 지난 주 대비 20% 증가' 같은 통계가 동기부여가 된다."

**복잡도**: M | **임팩트**: High | **우선순위**: P1

### 6.4 소셜 프루프 (Social Proof)

**기능 설명**: 방 목록에 사회적 증거를 추가하여 참여를 유도한다.

**구현 요소**:
- "친구 N명이 이 방에 있습니다" 표시
- "인기 방" 뱃지 (참여자 수 기준)
- "추천 방" 큐레이션 (편집팀 또는 알고리즘)
- 방 종료 후 "이 방은 N명이 참여했습니다" 공유 카드 생성

**사용자 스토리**: "학습자로서, 친구가 참여 중인 방을 보면 더 들어가고 싶다. '김철수님 외 3명의 친구가 참여 중' 표시가 있으면 바로 참여 버튼을 누르게 된다."

**복잡도**: S | **임팩트**: Medium | **우선순위**: P2

### 6.5 방 종료 후 요약 & 리뷰

**기능 설명**: 방이 끝난 후 참여 요약(대화 시간, 배운 단어, 획득 레몬)과 함께 방/호스트 리뷰를 남길 수 있는 화면.

**사용자 스토리**: "학습자로서, 방을 나갈 때 '오늘 25분 대화, 3개 새 단어, 5 레몬 획득' 요약을 보고, 호스트에게 별점과 한줄 평을 남기고 싶다."

**UI 구성**:
- 방 종료 시 자동 팝업
- 참여 시간 / 발언 시간
- 배운 어휘 목록 (어휘 공유 패널에서 저장한 것)
- 획득 레몬/XP
- 호스트 평가 (1-5 별점 + 선택적 코멘트)
- "다음 방 알림 받기" 옵션

**복잡도**: M | **임팩트**: High | **우선순위**: P1

## 7. 수익화 기회 (Monetization Opportunities)

### 7.1 프리미엄 방 기능

**기능 설명**: 무료 사용자는 기본 기능만, 프리미엄 구독자는 향상된 기능을 사용할 수 있다.

**사용자 스토리**: "진지한 학습자로서, 월 $4.99를 내고 녹음, 정기 방, 실시간 자막 기능을 사용하고 싶다. 이 기능들이 학습 효과를 크게 높여준다."

**복잡도**: L | **임팩트**: High | **우선순위**: P2

### 7.2 음성 대화방 전용 장비/아이템

**기능 설명**: 음성 대화방에서만 사용 가능한 특별 캐릭터 장비를 레몬으로 구매하거나, 업적 달성으로 획득한다.

**아이템 예시**:
- 골드 마이크 (프리미엄 스피커 이펙트)
- DJ 헤드셋 (호스트 전용 아이템)
- 빛나는 오라 (100시간 대화 달성)
- 특수 입장 이펙트 (방 참여 시 애니메이션)
- 커스텀 리액션 이모지

**사용자 스토리**: "학습자로서, 음성 대화방에서 내 캐릭터가 더 눈에 띄게 보이는 골드 마이크 아이템을 레몬으로 구매하고 싶다."

**복잡도**: M | **임팩트**: Medium | **우선순위**: P2

### 7.3 원어민 튜터 시스템

**기능 설명**: 인증된 원어민 튜터가 유료 1:1 또는 소그룹 레슨을 음성 대화방으로 진행할 수 있는 마켓플레이스.

**사용자 스토리**: "학습자로서, 인증된 한국어 원어민 튜터와 1:1 음성 레슨을 앱 내에서 예약하고 결제하고 싶다. 튜터의 프로필, 평점, 전문 분야를 확인하고 선택할 수 있다."

**복잡도**: XL | **임팩트**: High | **우선순위**: P3

## 9. 핵심 UI/UX 개선 포인트

### 9.1 현재 방 목록 화면 개선

**현재 문제점**:
- 단일 리스트 뷰만 제공, 필터/검색 없음
- 방 유형 구분 불가
- 원어민 여부 표시 없음
- 예약된 방 표시 없음

**개선안**:
```
[상단 탭: 실시간 | 예정 | 내 방]
[필터 바: 레벨 | 유형 | 태그 | 원어민]
[추천 섹션: "당신에게 맞는 방"]
[방 카드: 유형 아이콘 + 원어민 뱃지 + 태그 + 참여자 수]
```

### 9.2 현재 방 생성 화면 개선

**현재 문제점**:
- 방 유형 선택 없음
- 태그 추가 불가
- 예약 기능 없음
- 주제 카드 사전 설정 불가

**개선안**:
```
[Step 1: 방 유형 선택 (6종 카드 그리드)]
[Step 2: 기본 설정 (제목, 주제, 레벨, 스피커 수)]
[Step 3: 추가 설정 (태그, 예약 시간, 주제 카드 세트)]
[Step 4: 확인 & 생성]
```

### 9.3 현재 방 내부 화면 개선

**현재 문제점**:
- 스테이지(40%) + 채팅(30%) 고정 비율이 화면 공간 비효율
- 방 유형별 UI 분기 없음
- 어휘 공유, 퀴즈 등 교육 도구 없음
- 방 정보(주제, 참여자 정보) 접근이 제한적

**개선안**:
```
[AppBar: 방 제목 + 유형 뱃지 + 참여자 수]
[현재 주제 카드 (접을 수 있음)]
[스테이지 영역 (스와이프로 확대/축소)]
[탭 전환: 채팅 | 어휘 | 참여자]
[컨트롤바: 역할 기반 동적 버튼]
```

## 결론

현재 레몬 코리안의 음성 대화방은 탄탄한 기술 기반(LiveKit WebRTC, Flame 2D 스테이지, Socket.IO 실시간 통신)을 갖추고 있으나, **언어 학습에 특화된 기능**과 **사용자 재방문 유도 메커니즘**이 부족하다.

가장 시급한 개선 영역:
1. **방 유형 분류 + 발음 연습 방**: 한국어 학습이라는 본질적 가치 제공
2. **게이미피케이션 연동 (레몬 보상 + 스트릭)**: 참여 동기 부여
3. **호스트 도구 (주제 카드 + 어휘 공유)**: 학습 효과 극대화
4. **디스커버리 (원어민 표시 + 태그 + 스케줄)**: 적합한 방 찾기

이 4가지 영역에 집중하면, "오디오 소셜" 기능에서 "오디오 기반 한국어 학습 플랫폼"으로의 차별화를 달성할 수 있다. 특히 픽셀 캐릭터 스테이지는 경쟁 제품에 없는 독특한 요소로, 게이미피케이션과 결합하면 강력한 리텐션 드라이버가 될 수 있다.
